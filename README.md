# How to Build
1. Have Node.js installed.
2. Clone the repo and run:
```
git clone https://github.com/Ivanek333/test-playable.git
cd test-playable
npm ci
npm run build
```
3. The build output is a single html page located in the dist/ folder.
4. At the top of the page there is a json configuration, that controls most gameplay parameters.
